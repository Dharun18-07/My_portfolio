# My Personal Portfolio 🚀

This is my website where I show off my skills and the projects I've worked on. I built it for my User Interface Design class.

### 📂 What's inside?
* **index.html** - The home page with my profile picture.
* **about.html** - Information about my education, skills, and hobbies.
* **experience.html** - Details on my club memberships (ACM) and technical background.
* **projects.html** - A list of things I've built, like my Java apps and AI projects.
* **contacts.html** - How to reach me via email or phone.
* **style.css** - One file that handles all the colors, fonts, and layout.

### ✨ Features I Added
* **Mobile Friendly:** The website looks good on phones and laptops because the layout adjusts automatically by my responsive design.
* **Moving Background:** I added a smooth color-shifting animation to the background to make it look modern.
* **Project Cards:** Instead of just a list, my projects are organized into "cards" so they are easy to read.

### 🛠️ How I Built It
I used **HTML5** to structure the pages and **CSS3** to handle the design and animations. I made sure to use proper tags so the code is clean and easy to follow.
